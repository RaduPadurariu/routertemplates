CSS Intro

  • CSS is the language we use to style a Web page.
  • CSS stands for Cascading Style Sheets.
  • CSS describes how HTML elements are to be displayed on screen, paper, or in other media.
  • CSS saves a lot of work. It can control the layout of multiple web pages all at once.
  • External stylesheets are stored in CSS files.
There are three ways of inserting a style sheet (priority in this order):
  • Inline CSS - is used to apply a unique style for a single element. Is define inside HTML element using <style> tag.
  • Internal CSS (embed) - is used if one single HTML page has a unique style. Is defined in head section.
  • External CSS - is used to change the look of the entire website. Is defined in an external file with extension .css, and HTML pages must include in head the link tag for reference.

CSS Selectors

CSS simple selector - select elements based on name, id or class.

  • #id (#firstname) - Selects the element with id="firstname"
  • .class (.intro) - Selects all elements with class="intro"
  • element .class (p .intro) - Selects only <p> elements with class="intro"
  • * (*) - Selects all elements.
  • element (p) - Selects all <p> elements.
  • element, element (div, p) - Selects all <div> elements and all <p> elements.

CSS Background Image

  • background - sets all the background properties in one declaration (is called shorthand: {background: #ffffff url('img.png') no-repeat right top;})
  • background-image - sets the background image for an element. {background-image: url('img.png');}
  • background-size - specifies the size of the background images. {background-size: cover}
  • background-position - specifies the position for the background. {background-position: center or right top;}
  • background-repeat - sets how a background image will be repeated. {background-repeat: none, repeat-x or repeat-y;}
  • background-attachment - sets whether a background image is fixed or scrolls with the rest of the page. {background-attachment: fixed or scroll;}

CSS Box model

Box model is used in CSS when we talking about design and layout.

  • CONTENT - the content of the box, where text and images appear.
  • PADDING - clears an area around the content. Shorthand property: {padding: top right bottom left} or {padding:top and bottom left and right}
  • BORDER - a border that goes around the padding and content.
  • MARGIN - clears an area outside the border. Shorthand property: same as padding.
    The margin:auto value will center the element in its container. the margin:inherit value takes the margins from the parent element.
    Margin collapse: Only top and bottom margins of elements are sometimes collapsed into a single margin that is equal to the largest of the two margins.

CSS width and height

The CSS width and height properties are used to set the height and width of an element.
The CSS min-width, min-height, max-width, max-height are used to set min and max sizes of an element.
Properties width and height may have the following values: auto, length0 (px, cm, em, rem), %, initial, inherit.

CSS Outline

An outline is a line drawn outside the element's border. Unlike border, the outline is drawn outside the element's border, and may overlap other content. Also, the outline is NOT a part of the element's dimensions; the element's total width and height is not affected by the width of the outline.

  • outline - A shorthand property for setting outline-width, outline-style, and outline-color in one declaration.
  • outline-color - Sets the color of an outline.
  • outline-offset - Specifies the space between an outline and the edge or border of an element.
  • outline-style - Sets the style of an outline.
  • outline-width - Sets the width of an outline.

CSS Text

  • color - Specifies the color of text.
  • text-align (left, center, right, justify)- Specifies the horizontal alignment of text.
  • text-align-last - Specifies how to align the last line of a text.
  • direction (rtl, ltr)- Specifies the text direction/writing direction.
  • unicode-bidi (bidi-override)- Used together with the direction property to set or return whether the text should be overridden to support multiple languages in the same document.
  • vertical-align - Sets the vertical alignment of an element.
  • text-decoration - Sets all the text-decoration properties in one declaration.
  • text-decoration-color - Specifies the color of the text-decoration.
  • text-decoration-line - Specifies the kind of text decoration to be used (underline, overline, etc.).
  • text-decoration-style - Specifies the style of the text decoration (solid, dotted, etc.).
  • text-decoration-thickness - Specifies the thickness of the text decoration line.
  • text-transform - Controls the capitalization of text.
  • text-indent - Specifies the indentation of the first line in a text-block.
  • letter-spacing - Specifies the space between characters in a text.
  • line-height - Specifies the line height.
  • word-spacing - Specifies the space between words in a text.
  • white-space - Specifies how to handle white-space inside an element.
  • text-shadow - adds shadow to text specifying horizontal shadow, vertical shadow but also can add blur and color.

CSS Fonts

Is important to choose the correct color and text size for the font.
Those are the most important font families:

  • Serif - fonts have a small stroke at the edges of each letter. They create a sense of formality and elegance. Ex: Times New Roman, Georgia, Garamond.
  • Sans-serif - fonts have clean lines (no small strokes attached). They create a modern and minimalist look. Ex: Arial, Verdana, Helvetica, Segoe UI, Trebuchet MS.
  • Monospace - here all the letters have the same fixed width. They create a mechanical look. Ex: Courier New, Lucida Console, Monaco.
  • Cursive - fonts imitate human handwriting. Ex: Brush Script MT, Lucida Handwriting
  • Fantasy - fonts are decorative/playful fonts. Ex: Copperplate, Papyrus
The font-style property is mostly used to specify italic text.
The font-weight property specifies the weight of a font (bold).
The font-variant property specifies whether or not a text should be displayed in a small-caps font (small-caps).
The font-size property sets the size of the text. A great practice is to set the default font-size in percent for the body element: body {font-size:100%}, h1 {font-size:2.5em}, p {font-size:0.875em}.
The text size can be set with a vw unit, which means the "viewport width" to follow the size of the browser window. Ex: Resize the browser window to see how this font size scales.
